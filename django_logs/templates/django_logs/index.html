Discord Log Viewer

This website can be used to view logs uploaded to different services as they would appear on Discord.

Services we provide support for include:

  • Rowboat:
    • RowboatOGPrivate
    • AirplanePrivate
    • Aperture
    • FlygbåtPrivate
    • HeimdallrPrivate
    • HepBoat
    • JetskiPrivate
    • LMG ShowboatPrivate
    • RawgoatPrivate
    • SpeedboatPrivate

To use Rowboat logs, the URL pattern is {{ request.META.HTTP_HOST }}/rowboat/<filename>?type=<type>
Examples:

  • {{ request.META.HTTP_HOST }}/2b0rn0t2-b3th-at15-th3q-u35t10n12345?type=flygbåt
  • {{ request.META.HTTP_HOST }}/l0r3m1p5-umd0-l0r5-174m-37c0n53c737u?type=lmg

Alternatively, you can drop the URL in here, and we'll try and make the best of it.
